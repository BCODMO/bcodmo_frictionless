# BCO-DMO Pipeline Processors Summary

This file provides a quick reference for all available processors. See README.md for detailed parameter documentation.

## Custom BCO-DMO Processors

### Loading & Output
- `bcodmo_pipeline_processors.load` - Load data from files (CSV, Excel, fixed-width, etc.) into the pipeline
- `bcodmo_pipeline_processors.concatenate` - Combine multiple resources into a single resource with field mapping
- `bcodmo_pipeline_processors.dump_to_path` - Save data to a local filesystem directory
- `bcodmo_pipeline_processors.dump_to_s3` - Save data to S3-compatible storage

### Field Manipulation
- `bcodmo_pipeline_processors.rename_fields` - Rename fields using explicit old/new name pairs
- `bcodmo_pipeline_processors.rename_fields_regex` - Rename fields using regex find/replace patterns
- `bcodmo_pipeline_processors.reorder_fields` - Change the order of fields in a resource
- `bcodmo_pipeline_processors.update_fields` - Update field metadata (title, description, etc.)
- `bcodmo_pipeline_processors.set_types` - Set field types and type options (number, string, date, etc.)
- `bcodmo_pipeline_processors.add_schema_metadata` - Add metadata to the resource schema

### Data Transformation
- `bcodmo_pipeline_processors.boolean_add_computed_field` - Create new fields using boolean conditions and math expressions
- `bcodmo_pipeline_processors.boolean_filter_rows` - Keep only rows that match a boolean condition
- `bcodmo_pipeline_processors.find_replace` - Find and replace text using regex (supports uppercase/lowercase functions)
- `bcodmo_pipeline_processors.string_format` - Create formatted strings from multiple fields using Python format syntax
- `bcodmo_pipeline_processors.split_column` - Split a field into multiple fields using regex or delimiter
- `bcodmo_pipeline_processors.edit_cells` - Edit specific cells by row number
- `bcodmo_pipeline_processors.extract_nonnumeric` - Move non-numeric values from a field to a new field
- `bcodmo_pipeline_processors.round_fields` - Round numeric fields to specified decimal places
- `bcodmo_pipeline_processors.convert_units` - Convert between units (feet→meters, fathoms→meters, etc.)

### Date & Coordinate Processing
- `bcodmo_pipeline_processors.convert_date` - Convert dates between formats (Python strftime, Excel serial, MATLAB, decimal day/year)
- `bcodmo_pipeline_processors.convert_to_decimal_degrees` - Convert coordinates from degrees-minutes-seconds to decimal degrees

### Resource Management
- `bcodmo_pipeline_processors.remove_resources` - Remove resources from the pipeline
- `bcodmo_pipeline_processors.rename_resource` - Rename a resource
- `bcodmo_pipeline_processors.join` - Join two resources with aggregation support

## Standard Dataflows Processors

- `duplicate` - Create a copy of a resource
- `update_package` - Update package-level metadata
- `update_resource` - Update resource-level metadata
- `delete_fields` - Remove fields from resources (supports regex)
- `sort` - Sort rows by field values
- `add_computed_field` - Add fields using operations (sum, avg, max, min, multiply, join, format, constant)
- `unpivot` - Transform columns to rows (wide to long format)
- `join` - Join two resources together with aggregation

## Boolean Syntax (for boolean_add_computed_field, boolean_filter_rows, and boolean_statement parameters)

Terms: `{field_name}`, `'string'`, `123`, `null`, `LINE_NUMBER`, `re'regex'`
Operators: `==`, `!=`, `>`, `>=`, `<`, `<=`, `AND`, `OR` (or `&&`, `||`)
Example: `{depth} > 100 AND {species} != null`

## Math Syntax (for boolean_add_computed_field with math_operation=true)

Operators: `+`, `-`, `*`, `/`, `^`
Example: `{field1} + {field2} * 2`
